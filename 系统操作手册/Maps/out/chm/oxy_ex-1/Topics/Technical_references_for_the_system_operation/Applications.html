应用程序

应用程序设置下，可以查看服务设置、远程配置、用户设置和控制面板设置。

  1. 进入Service->Configuration 界面。
  2. 从左侧导航栏中选择Service Setting ，显示Screen Shot, FTP, Tenet 等设置选项。只可查看，不可修改。
  3. 从左侧导航栏中选择 Remote Config，进入远程配置界面。在导航菜单选择 Remote Config，显示远程配置相关选项。若将上传服务从开启调整至关闭，会删除缓存在数据 库中的待上传数据和正在上传数据。
  4. 从左侧导航栏中选择 User Setting，可以查看用户设置相关内容。
  5. 从左侧导航栏中选择 Panel Configure，进入 Panel Config 页面，配置各界面中控制面板的位置。选中 Panel（Left），则控制面板显示在界面你左侧，反之，显示在右边。
